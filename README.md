## :clipboard: 개요

"SpringBoot + React 기반 온라인 합주 플랫폼"

- 개발 기간 : 2023. 08. 21. ~ 2023. 10. 06.

## 💡 핵심 기능

- 소셜 로그인(카카오, 구글)
- 솔로 연주 & 합주 녹화 서비스
- 키워드에 따른 검색 시스템
- 선호하는 악기, 유저별 취향 영상에 따른 추천 시스템
- SNS (댓글, 팔로잉, 실시간 알림 서비스)

## 🧑‍💻 추천 요소

- 선호하는 악기(회원가입)
- 선호하는 장르(회원가입)
- 시청한 영상(활동데이터)
- 좋아요한 영상(활동데이터)
- 업로드한 영상(활동데이터)
